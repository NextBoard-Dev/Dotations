ARCHIVAGE LOCAL DES PDF SIGNES

Structure prevue :

- `archives/pdf/arrivee/`
- `archives/pdf/sortie/`
- `archives/metadata/`

But :

- stocker les futurs PDF signes
- preparer une arborescence simple et stable
- garder une logique proche du futur stockage heberge

Regle cible :

- un document d'arrivee ira dans `archives/pdf/arrivee/`
- un document de sortie ira dans `archives/pdf/sortie/`
- les metadonnees d'indexation iront dans `archives/metadata/`

Exemple de nom de fichier :

- `2026-03-02_DUPUIS_LAURENT.pdf`
