# Smartphone Dotations (Vite + React + Supabase)

## Lancer en local

1. Ouvrir le dossier `smartphone`.
2. Installer les dependances: `npm install`
3. Lancer l'application: `npm run dev`

## Build statique

- Generer le build: `npm run build`
- Les fichiers generes sont dans `smartphone/dist`.

## Variables d'environnement

Configurer si besoin dans `.env.local`:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```
