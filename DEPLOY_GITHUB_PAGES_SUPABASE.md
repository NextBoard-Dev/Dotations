# DEPLOIEMENT GITHUB PAGES + SUPABASE (DOTATIONS)

Date de mise a jour: 16/03/2026

## 1) Pre-requis

- Repo GitHub: `Mililumatt/Dotations`
- Branche: `main`
- Source Pages: `main / root`

## 2) Checklist avant push

- Pas de dossier `snapshots/` versionne.
- Les fichiers modifies correspondent uniquement au besoin.
- Les regles metier critiques sont validees:
  - blocage PDF sans double signature,
  - archives uniquement documents signes,
  - filtrage references par type + site.

## 3) Publication

1. Commit local (GitHub Desktop).
2. Push `origin/main`.
3. Verifier l'URL publique GitHub Pages.

## 4) Verification post-deploiement

- Fiche Personne:
  - ajout/modif/suppression effet,
  - champs contextuels grises et stables.
- Documents:
  - signatures PC/mobile,
  - blocage ouverture PDF si signatures manquantes.
- Archives:
  - presence uniquement des documents valides.
